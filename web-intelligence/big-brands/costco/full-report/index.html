Vendor Bake-Off Readiness
Can AI compare Costco head-to-head when asked?

Bake-Off Absent

All three readiness checks fail. Pricing extractability: the /membership landing page hides pound figures behind a "Join" CTA. Although Bing's top snippet confirms "starting from £15" lives on some Costco URL, the canonical landing page does not surface it in extractable HTML text. Feature data structure: bullet lists appear, but no <table> or <dl> elements that would let AI build a comparison row. Comparison claims: marketing copy uses adjective-only framing ("great savings", "quality you can trust") with no factual differentiators against Bestway, Sam's Club, or supermarket bulk-buying.

AI Crawler Access
Can AI engines read costco.co.uk?

Fully open

None of GPTBot, ClaudeBot, anthropic-ai, OAI-SearchBot, PerplexityBot, GoogleOther, or CCBot are blocked. The wildcard User-agent: * block disallows /checkout, /my-account, paginated category, and cart subtotal endpoints only. None of those paths cover report-relevant content. Bingbot and Bravebot are also unrestricted. This is the optimal AI-crawler configuration.

Multi-Engine Indexation
Is Costco visible across the AI search substrate, not just Google?

Partial

Bingbot and Bravebot both allowed in robots.txt (no per-bot restrictions). IndexNow not detected at canonical key-file paths and no x-indexnow header on the homepage. Sitemap is declared at http://www.costco.co.uk/sitemap.xml. The two standard recommendations below close to Multi-engine ready.

Standard recommendations

1. Verify in Bing Webmaster Tools at bing.com/webmasters. Submit sitemap. Bing's index powers ChatGPT search and ChatGPT browse mode. Roughly 30 minutes.

2. Implement IndexNow. Generate key, host at /<key>.txt, ping endpoint on every publish. One call notifies Bing + Yandex + Seznam + Naver instantly. Roughly 1 hour.

Brave note: No webmaster tools exist. Bravebot status above is the only controllable surface. Brave's index is otherwise fed by the Web Discovery Project (passive opt-in via Brave-browser users) plus a Bing-licensed fallback for sparse queries.
