Multi-Engine Indexation
Are you visible across the AI search substrate, not just Google?

Partial

Bingbot: allowed · Bravebot: allowed · IndexNow: not detected. A check on /indexnow-key.txt returned HTTP 400 (key file absent) and no x-indexnow response header was found in the site headers. Without IndexNow, new content and updates wait days for Bing's crawler to re-index them — slowing any improvements you make from reaching ChatGPT search and ChatGPT browse mode users. Brave has no webmaster tools; Bravebot status (allowed) is the only controllable surface. Note on Bing's 0/6 score: the Bing zero is not an indexation gap — it is a tokenisation pattern. Bing returns dictionary definitions or wrong-vertical content for natural-language multi-word queries in niche verticals. The fix is literal-phrase H2 headings and exact-match page titles, not crawl access.

Standard recommendations

1. Verify in Bing Webmaster Tools — bing.com/webmasters. Claim the property, submit the sitemap at leogym.co.uk/sitemap.xml, and run URL-inspect on your homepage, HYROX page, and class pages. Bing's index powers ChatGPT search and ChatGPT browse mode. ~30 min.

2. Implement IndexNow — generate a key, host the key file at /<key>.txt, and ping the IndexNow endpoint on every content publish. One call notifies Bing, Yandex, Seznam, and Naver instantly. Without it, new class descriptions or staff updates wait for the next crawl cycle. ~1 hour setup.

Brave note: No webmaster tools exist for Brave Search. Bravebot is currently allowed in robots.txt — that is the only controllable surface. Brave's index is fed by Bravebot crawling plus the Web Discovery Project (passive opt-in via Brave-browser users). IndexNow does not notify Brave.

Your next move
Add gym structured data across all class pages
None of the 18 tested cells detects schema markup (citation factor 4) anywhere on leogym.co.uk — it is the single most-missed factor in this audit. Adding GymOrHealthClub, SportsActivityLocation, and FAQPage structured data to your homepage, individual class pages, and the HYROX page gives AI engines a direct, machine-readable signal about what you offer, where you are, and who the answer is for. For Google, schema is a direct eligibility signal for AI Overviews; for Brave, it boosts placement in AI Answer results. Pair this with adding named coach credentials (qualifications, certifications, years of experience) to each class and instructor page, which unlocks citation factor 2 across all query types. Effort: approximately 2 hours with a developer or an SEO plugin if you are on a CMS that supports it.
