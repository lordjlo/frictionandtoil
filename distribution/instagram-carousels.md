## Production rules
- One cover frame per piece, brand-consistent across all four carousels (charcoal + gold + serif).
- All quotes set in Cormorant Garamond italic, 56–72pt depending on length.
- All stats set in JetBrains Mono, monospace alignment, source line beneath in 14pt.
- Final slide always carries the URL in clean mono — no shortened links.
- Stories: cover frame of each carousel goes to Stories the day the post lands, with "swipe up" sticker pointing to the article.
